Data Bases

PostgreSQL is probably my favorite data base system. PostgreSQL is a powerful, open source object-relational database system. I've been using it for the last 8 years and will keep doing it every time I can.

I've used MySQL for 4 years for different products, working for WebExperto and still do for some of my current projects. I'm impressed of its high performance and ease of use. Anyway, I still choose PostgreSQL these times.

I started to use mongoDB with Django this year and I'm really impressed on it's high performance.
